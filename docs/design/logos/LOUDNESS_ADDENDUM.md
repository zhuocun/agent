# Olune Logo — Loudness Addendum (v3)

> **Amendment to** [`RICHNESS_CHARTER.md`](./RICHNESS_CHARTER.md). Defines HOW an
> Olune mark becomes **louder** without relaxing any charter clause. v2 was rejected
> as **still too quiet**.

---

## Why v2 reads quiet

1. **Figure too small** — marks float at ~50% of frame; excessive Ma reads as timid.
2. **Weight too thin** — monoline slivers and light phase splits.
3. **Value steps too close** — `pale` field ≈ `#F9FAFC` canvas; depth doesn't register.
4. **Mid-value brand on near-white** — `#2079E8` lacks punch against `#F9FAFC`.

Quiet is **not** fixed with a second hue or effects on exported marks.

---

## Loudness levers (measurable)

| Lever | Requirement | Charter basis |
| --- | --- | --- |
| **Scale** | Figure ≥ **75%** of live area (maskable safe-zone respected) | §1 layered geometry; HIG depth |
| **Weight** | Stroke ≥ **88/512** or **solid filled mass**; no thin slivers | §1 heavier stroke, fuller fills |
| **Value contrast** | ≥2 ramp steps, OKLCH **ΔL ≥ 0.14** (`brand` 0.59 vs `deep` 0.45). `pale` is canvas-side only | §2 ramp |
| **Dark luminous parity** | Navy `#080B11` + light step `#5BA0F2` is **mandatory hero** per concept | §6.8 theme parity |
| **Atmosphere** | Welcome/loading lockups only — dual-hue bloom, halo behind mark. Mark stays flat single-hue | §5 static-vs-motion |

---

## Hard ceiling (negation block — copy into every export prompt)

```
NO gradient, NO blend/fade, NO glow, NO drop-shadow, NO bevel, NO 3-D, NO glass,
NO second saturated hue on the mark. Louder = scale + weight + value-contrast
(+ welcome-only atmosphere on canvas). Louder is NEVER an effect on the exported mark.
```
